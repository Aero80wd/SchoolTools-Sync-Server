## 使用方式
### 安装依赖库
请先使用以下命令安装依赖库：
pip install flask
### 使用
将项目克隆下来后，运行根目录中的`main.py`，即可启动服务器
> **注意**
> 项目默认端口为`5000`，如有端口冲突请修改`main.py`中的运行端口。
